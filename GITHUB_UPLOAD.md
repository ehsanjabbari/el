# راهنمای آپلود در GitHub Pages

## مشکل حل شد! ✅

فایل `index.html` ایجاد شد. حالا می‌توانید پروژه را در GitHub آپلود کنید.

## فایل‌های مورد نیاز برای آپلود:

1. **index.html** ← فایل اصلی (حتماً آپلود کنید)
2. **styles.css** ← فایل طراحی
3. **app.js** ← فایل جاوااسکریپت
4. **README.md** ← راهنمای استفاده

## مراحل آپلود:

1. فایل `index.html` را به repository خود اضافه کنید
2. فایل‌های `styles.css` و `app.js` را نیز آپلود کنید
3. در Settings → Pages، Source را روی "Deploy from a branch" قرار دهید
4. Branch را روی "main" قرار دهید و فولدر را روی "/ (root)" بگذارید
5. چند دقیقه صبر کنید تا GitHub Pages فعال شود

## نکته مهم:
فایل `index.html` حتماً باید در ریشه repository باشد (نه در پوشه) تا GitHub آن را به عنوان صفحه اصلی اجرا کند.

## بعد از آپلود:
آدرس شما به شکل زیر خواهد بود:
`https://yourusername.github.io/repository-name`

اکنون سیستم حسابداری به درستی اجرا خواهد شد!
